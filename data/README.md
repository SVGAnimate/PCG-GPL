## Synopsys : version
`version ::= major['_' minor['_' patch]];`

 - `major ::= 2014/2099;` (use a valide year)
 - `minor ::= 01/12;` (use a valide moth)
 - `patch ::= 01/31;` (use a valide day)

## Format
 - `version.json`,
 - `version.xml`,
 - `version.php`,
 - `version.js`,
 - `version.html`;

## Sample
 - Release : 2026.json (Le Millésime, très recherchée et appréciée par les amateurs.)
 - Revision : 2026_01.json
 - Patch : 2026_03_18.json
